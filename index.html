0一句话结论

Telegram

有大量 DramaBox / ReelShort / GoodShort / ShortMax 等「全集分享」频道，英字/英配为主；原生英文与中文剧常混在一起。多为未授权搬运，封号、钓鱼、木马风险高，不适合当稳定正规片源。

免费 / 订阅

主流短剧 App 基本是前几集免费 + 金币。相对更「尽量免费」的是 PineDrama（TikTok） 与 YouTube；想「付一次尽量多看」优先核对 DramaBox / ReelShort 订阅权益原文，别默认 VIP = 全站无限。

1.4 风险（务必读）

  • 版权：绝大多数是未授权全集搬运，随时被封。
  • 安全：外链盘、APK、「付费进 VIP 群」中木马 / 钓鱼常见。
  • 质量：残缺合集、错剧名、低清、广告机器人。
  • 工程价值：不适合作为合规、可自动化的稳定上游片源。

2全网：更接近「全免费」的平台

竖屏短剧商业模型普遍是「前几集免费 + 金币 / 周卡」。几乎没有「与 ReelShort 同量级片库、永久全站 0 付费」的正经平台。

平台 模式 英文 / 英配 说明
PineDrama
TikTok 官方短剧 App		 现阶段主打免费、少广告 / 无付费墙 有英文区内容 用户评价常见 “no paywall”；不保证永久（增长后可能改收费）。目前最值得先试的「尽量免费」官方选项之一。
YouTube 免费 + 广告 英字 / 英配频道多 官方号常放片段引流 App；也有整部上传。可搜剧名 + full / English dub。部分频道后半段要 Channel Membership。
FreeShort 等「Free」字号 App 宣称 free 有 名称 ≠ 真全免费，需看是否后期锁集。
Dailymotion 等 UGC 免费 零散整剧 搜剧名即可；质量与版权参差。
Bilibili 国际版等 部分免费 英字中长剧更多 更偏常规剧，不完全是竖屏 micro-drama。
若目标是「尽量正规 + 少付费」：优先试 PineDrama，再用 YouTube 补剧名搜索。

容易踩坑的「伪免费」

  • App 名带 Free，实际仍锁后半部
  • Web「全集站」（如部分宣称 free full episodes 的站点）：版权与可持续性不明，宜当灰产站看待
  • 网上「免费看全集教程」大量指向盗链 / 搬运，有法律与账号风险

3「订阅无限看」类平台

价格与权益会变，下表为公开报道 / 对比文中的量级参考，以下载 App 内权益页为准。

平台 大致模式 备注
DramaBox 周卡 / 年卡 + 金币；有文写约 $5.99/周 或 ~$50/年 量级 相对更推订阅；是否「真无限全库」以 App 为准
ReelShort VIP 偏贵（有文写约 $19.99/周 或 ~$200/年）+ 强金币 原生英文内容多；按币买全集很贵
MyDrama 等 App + YouTube 会员看续集 更像「会员解锁后续集」
ShortMax / GoodShort / NetShort 金币为主 + 订阅 / 充值入口 GoodShort 实测多为按集金币；会员别默认 = 全站

开订阅前核对三句话

  1. 是全站无限，还是折扣 / 去广告 / 每日送币？
  2. 是否包含新剧 / 热剧？
  3. 你所在地区是否有版权限制？
要大量看、不想一集一集充 → 优先比 DramaBox 类年 / 周订阅。
要原生英文 → ReelShort 内容强，但 VIP 往往更贵。
